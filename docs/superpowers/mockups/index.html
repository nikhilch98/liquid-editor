Approved design mockups

Static HTML mockups rebuilt from the approved spec (docs/superpowers/specs/2026-04-18-liquideditor-premium-ui-design.md). The original .superpowers/brainstorm/ HTML mockups referenced in the spec were not committed to the repo, so these are a reconstruction pinned to the textual design tokens (hex values, spacing, typography) verbatim. Open each in any browser; no JS, no external assets.

  • Editor — iPhone portrait Chrome, preview, transport, timeline, tool shelf (§3.1).
  • Project Library — iPhone Header, search, recent rail, grid, FAB, 4-tab bar (§4.1).
  • Export — iPhone Ring, preset segmented, steppers, toggles, sticky CTA (§5.2).
How to use: open these alongside the running app to compare. Hex tokens in each file exactly match LiquidColors in the codebase. If the app doesn't look like the mockups, the screen is missing a token or using a legacy one.
